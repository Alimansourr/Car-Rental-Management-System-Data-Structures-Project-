# Car Rental Management System (C++)

A console-based car rental management system created as a Data Structures project.

##  Features
- User signup & login
- Store user records in files
- Add, delete, update users
- Sort users by last name (Insertion Sort - Linked List)
- Rent cars with date validation
- Track current and past rentals
- Save data back to files

##  Technologies
- C++
- File Handling (`ifstream`, `ofstream`)
- Singly & doubly linked lists
- Date comparison utilities

##  How to Run
1. Open the project in your IDE (CodeBlocks / VS / Dev C++).
2. Compile `main.cpp`.
3. Make sure the files exist in the same folder:
   - `users.txt`
   - `cars.txt`
   - `rented_cars.txt`
4. Run the program.
